Claude 技巧大全！6个独家技巧，解锁Claude潜能(附 8 个实用示例)

AI提示词

一个好的提示词和坏的提示词意见的差距非常大。

好的提示词就像是一把钥匙，引导AI模型为我们提供精准、有针对性的回答。如果提示词使用不当，就像是用错了钥匙，结果往往是答非所问，甚至让人哭笑不得，需要花费很多的时间进行多轮对话才有可能得到最终我们想要的结果。

因此，我们在与大语言模型交流时，一定要学会运用恰当的提示词，这样才能让模型更好地理解我们的意图，从而获得更满意的答案。

为此，如何写一段好的提示词也是我们运用好Claude非常重要的课题。

最好的方式也可以借鉴之前总结好的优秀的提示词，比如 程序员精选常用ChatGPT提示词合集

Claude

Claude3.5是Anthropic公司推出的大语言模型，具备卓越的性能提升，特别是在推理、知识掌握和多语言处理方面。它运行速度更快，成本更低，理解能力更强，并引入了创新的Artifacts功能，实时互动生成内容。此外，其定价策略灵活，提供不同级别的服务，满足不同用户需求，展现了其在AI领域的领先地位。

目前Claude和ChatGPT在各个方面都有自己擅长的方向，针对具体的工作使用好对应AI模型也是非常重要的。

根据 Claude 3.5订阅升级教程 ，可以快速注册和升级Claude，可以无限制使用最新版本Claude 3.5 Sonnet模型。

Claude提示词的一般技巧

一个好的提示词往往具有下面这些特点：

  • 清晰具体
  • 使用示例
  • 鼓励思考
  • 迭代改进
  • 利用Claude的知识
  • 使用角色扮演

清晰具体

一般情况下，满足下面三个要求才能算得是清晰具体

  • 在消息开头清楚陈述任务或问题
  • 提供背景和细节，帮助Claude理解您的需求
  • 将复杂任务分解为更小、可管理的步骤

使用示例

如果你需要Claude回答的问题的输出是一个固定结构，那么最好是给Claude提供对应的使用示例。

  • 提供您期望的输出类型的示例
  • 如果您需要特定格式或风格，向Claude展示一个示例

鼓励思考

对于复杂任务，要求Claude”逐步思考”或”解释推理过程”，这可以导致更准确和详细的回应

迭代改进

如果Claude的第一个回应不太正确，请求澄清或修改，您可以说 这很接近，但能否调整X更像Y？

利用Claude的知识

Claude拥有广泛的知识。不要犹豫，请求解释或背景信息，确保包括相关的上下文和细节，使Claude的回应最有针对性

使用角色扮演

角色扮演估计是目前最常见的提示词技巧

要求Claude在回应时采用特定的角色或视角，这可以帮助获得更全面和有洞察力的回答

Claude特定任务的技巧和示例

上面的提示词是通用技巧。但是面对不同类型的任务，对应的提示词也需要进行相应的调整和优化。

使用Claude内容创作提示词

  • 指定你的目标受众
  • 定义所需的语气和风格
  • 提供输出结构的框架

使用Claude文档摘要和问答

  • 明确指出你想要什么
  • 使用文档名称进行引用
  • 要求Claude引用文档中的特定部分

使用Claude数据分析和可视化

  • 指定所需的数据格式
  • 明确要分析的关键指标
  • 要求提供数据驱动的建议

使用Claude头脑风暴

  • 要求Claude生成一系列可能性或替代方案
  • 指定要覆盖的具体主题
  • 请求以特定格式（如要点、编号列表或表格）呈现响应

使用Claude开发营销策略

  • 指定目标受众和产品
  • 要求市场分析、目标受众画像、营销组合等具体内容
  • 请求结构化的输出格式

使用Claude分析财务报告

  • 提供具体的财务报告文件
  • 要求以特定角色（如CFO）进行分析
  • 指定分析的具体方面和所需的输出结构

Claude提示指南减少胡说八道

大语言模型目前很大的问题就是，有时候面对他不知道的问题，就会一本正经的胡说八道。如果我们缺乏针对这个问题的相关背景知识，就完全会被大语言模型误导。

所以，如何有效避免Claude胡说八道，最大限度的减少Claude幻觉，可以有效提高我们使用Claude的回答问题的准确性。

下面就为大家提供一些可以减少Claude胡说八道的技巧。

允许CLAUDE承认不确定性

告诉Claude如果不确定可以直接说不知道

例如：“如果你不确定某事，可以直接承认。只需说你不知道即可。”

分解复杂任务

如果任务看起来太大，Claude可能会遗漏步骤或某些步骤表现不佳，将任务分解为更小的步骤，一次一条消息地与Claude一起完成这些步骤

包含所有上下文信息

Claude不会保留之前对话的信息，在每个新对话中包含所有必要的上下文

性能最大化技巧

  • 使用清晰、具体的语言
  • 提供丰富的上下文和背景信息
  • 使用结构化的提示格式（如列表或大纲）
  • 对于复杂任务，要求逐步思考
  • 利用Claude的跨领域知识
  • 鼓励创意思考和问题解决

最小化幻觉的策略

要求Claude在不确定时表明，请求引用或解释推理过程，对关键信息进行交叉验证

注意可能表明不确定性的措辞，对于事实性问题，考虑多次询问以检查一致性

使用Claude的8个经典实例

清晰具体的任务描述

我需要帮助创建一个10张幻灯片的演示文稿，用于我们的季度销售会议。演示文稿应涵盖我们的第二季度销售业绩、最畅销产品和第三季度的销售目标。请提供一个大纲，包括每张幻灯片的要点。

这个提示词就明确了三点：

  • 明确指定了幻灯片数量
  • 提供了具体的主题和内容要求
  • 要求提供结构化的输出（大纲）

使用示例和角色扮演

作为一名高级营销顾问，我需要你的帮助为我们的新型环保智能手机配件系列制定一个全面的营销策略。我们的目标受众是环保意识强的千禧一代和Z世代消费者。请提供一个详细的策略，包括：1. 市场分析
2. 目标受众画像
3. 营销组合（4P）
4. 内容策略
5. KPI和测量方法

请以结构化格式呈现这些信息，并在相关处解释你的推理或提供简短示例。

  • 使用角色扮演增强回答的专业性
  • 提供了详细的背景信息
  • 要求结构化输出和解释推理

复杂任务分解

我正在寻找提高团队生产力的方法。请逐步思考这个问题，考虑以下因素：1. 当前的生产力障碍（例如，会议过多、优先级不明确）
2. 潜在的解决方案（例如，时间管理技巧、项目管理工具）
3. 实施挑战
4. 衡量改进的方法

对于每一步，请简要解释你的推理过程。最后，总结你的想法。

  • 将复杂问题分解为具体步骤
  • 要求逐步思考和解释推理
  • 包括总结请求

数据分析和可视化

我已附上一个名为’2023年销售数据.xlsx’的电子表格。请分析这些数据并以以下格式呈现关键发现：1. 执行摘要（2-3句话）
2. 关键指标： - 每个季度的总销售额

  • 表现最佳的产品类别
  • 增长最快的地区
  1. 趋势： - 列出3个值得注意的趋势，每个都附带简短解释
  2. 建议： - 提供3个基于数据的建议，每个都附带简短理由

分析完成后，请建议三种能有效传达这些发现的数据可视化类型。

  • 明确指定了数据源和分析格式
  • 要求具体的指标和见解
  • 包括数据可视化建议请求

多角度分析与对比

作为一位经验丰富的政策分析师，请对’普及高等教育’这一政策进行全面分析。请从以下几个角度展开：1. 经济影响：短期成本 vs 长期收益
2. 社会影响：社会流动性、就业市场变化
3. 教育质量：可能面临的挑战和机遇
4. 国际竞争力：在全球人才市场中的地位变化

对于每个角度，请提供：- 潜在的积极影响

  • 可能的负面后果
  • 相关的国际案例或研究数据
  • 对可能的质疑和批评的回应

最后，基于您的分析，提出2-3个平衡各方利益的政策建议。

  • 要求多角度、深入的分析
  • 鼓励批判性思考和平衡观点
  • 结合理论分析和实际案例

创意写作与结构化输出

请以一名时间旅行者的身份，创作一篇短篇科幻故事。故事应该包含以下元素：1. 设定：2200年的地球和3000年的火星殖民地
2. 主角：一位来自2023年的历史学家
3. 冲突：主角意外改变了历史，必须修复时间线
4. 主题：人类进步与环境责任的平衡

请按照以下结构呈现故事：- 开篇（100字）：介绍背景和主角

  • 发展（200字）：描述时间旅行和意外改变历史的事件
  • 高潮（150字）：主角努力修复时间线的关键时刻
  • 结局（50字）：故事的结局和主题的体现

每个部分后，请简要说明你选择的关键词或短语如何强化故事主题和氛围。

  • 结合创意写作和结构化输出
  • 提供详细的故事元素和结构指南
  • 要求对创作过程进行分析

技术文档与用户体验

我们正在开发一款新的智能家居应用程序，需要创建用户文档。请按照以下步骤帮助我们：

  1. 创建一个’快速入门指南’的大纲，面向非技术用户。包括：

    • 应用安装

    • 设备连接

    • 基本功能设置

  2. 为’常见问题’部分设计5个最可能的问题和答案。

  3. 编写一个详细的步骤说明，演示如何设置自动化规则（例如，当检测到动作时打开灯）。使用简单的语言和具体的例子。

  4. 创建一个技巧和诀窍部分，包括5个提高用户体验的建议。对于每个部分，请考虑：

    • 如何使用视觉元素（如图标或截图）来增强理解

    • 可能的可访问性考虑（如色盲用户）

    • 如何使用视觉元素（如图标或截图）来增强理解

    • 如何整合用户反馈机制

  5. 最后，提供3-5个衡量这份文档有效性的指标建议。

  • 多层次任务，覆盖不同类型的文档
  • 集中于用户体验和可接力性
  • 要求考虑视觉元素和效果评估

跨学科问题解决

作为一个跨学科专家团队的协调者，我们正在研究如何改善城市的空气质量。请从以下几个学科的角度来分析这个问题并提出解决方案：1. 环境科学：分析主要污染源和其影响
2. 城市规划：提出有助于改善空气质量的城市设计方案
3. 交通工程：建议可持续的交通系统改进
4. 经济学：评估不同解决方案的成本效益
5. 公共卫生：探讨改善空气质量对公共健康的潜在影响
6. 政策与法律：提出可行的政策和法规框架

对于每个学科的分析：- 提供一个关键的见解或发现

  • 推荐一个具体的行动或解决方案
  • 指出该方案可能面临的挑战或限制

最后，整合这些跨学科的见解，提出一个全面的、多方位的行动计划，以在10年内显著改善城市空气质量。包括短期和长期目标，以及如何衡量进展。

  • 要求高度跨学科的分析和综合
  • 鼓励从多个角度思考复杂问题
  • 要求提出全面的、可操作的解决方案
